=== WooCommerce Bambora Gateway ===
Author: skyverge
Tags: woocommerce
Requires PHP: 5.6
Requires at least: 4.4
Tested up to: 5.4.2

== Description ==

Adds the Bambora Payment Gateway to your WooCommerce site.

See http://docs.woocommerce.com/document/bambora/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-beanstream' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
